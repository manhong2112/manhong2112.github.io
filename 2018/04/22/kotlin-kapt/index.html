    最近在做 Mimikko-Mod, 想加個 Xposed 的 Annotation 自動 hook // 就是挖坑給自己跳

    部分內容從簡, 因為我忘了

    首先呢, 照着以前 Android apt 的教程（畢竟我根本沒找到 Kotlin + Android + Annotation processor 的詳細教程), 弄了兩個 module, annotation和processor…

    root/settings.gradle

    annotation/build.gradle

    processor/build.gradle

    app/build.gradle

    然後真正弄個 annotation class..

    annotation/../Hook.kt

    然後 annotation processor…

    processor/../HookProcessor.kt

    嗯..至此一切正常, 因為不正常的都跳過了

    然後要測試@SupportedAnnotationTypes, 於是註解掉throw Throwable("init")…

    喵喵喵??? 這個不應該是 @AutoService 處理的嗎?
    翻了翻processor/build, 不對啊, javax.annotation.processing.Processor 有生成啊, me.manhong2112.xposed.processor.HookProcessor

    也在裡面試着複製出來塞到 main 裡, 錯誤依舊

    求助Google

    SO 某用戶指出可以

    嗯…管用就行了~

    最後確保 process 管用, 隨便找一個 method 加上 @HookAfter("")

    至此 annotation processor 終於是能用了
